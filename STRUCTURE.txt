🎯 Key Features
───────────────

✅ Sidebar Layout
   - Left sidebar with profile info, contact, navigation
   - Sticky on desktop, responsive on mobile
   - Profile photo, name, title, email, phone links

✅ Employment Filtering
   - Type: Technical, Customer Service, Childcare
   - Time Period: Current, 1yr, 3yr, 5yr
   - Reset to defaults (Technical only shown)
   - Client-side JavaScript filtering

✅ Responsive Design
   - Desktop: Sidebar + content
   - Tablet: Single column with sidebar
   - Mobile: Stacked layout, touch-optimized

✅ Two-Tier Skills
   - Technical: Languages, frameworks, tools
   - Professional: Communication, management, incident response, etc.

✅ GitHub Integration
   - Personal projects link to repos
   - What/Why/Learnings/Futures format
   - Easy to add new projects


📝 Employment Format
────────────────────

Each employment file has frontmatter:

---
title: "Job Title"
company: "Company Name"
location: "City, State"
start_date: "Jan 2023"
end_date: "Present"
type: "technical"              ← technical | customer-service | childcare
period: "current"              ← current | 1year | 3years | 5years
---

Content with markdown...

## Responsibilities & Achievements
- Bullet point

## What I Gained
- Learning point


🔍 Filtering Logic
────────────────────

Type Filter (Checkboxes):
- technical
- customer-service
- childcare

Period Filter (Dropdown):
- "" (empty = all)
- current
- 1year
- 3years
- 5years

Default: Shows technical entries only
JavaScript updates display as filters change

🚀 Deployment Steps
────────────────────

1. Update _config.yml with your info
2. Add profile photo
3. Customize pages with your content
4. Test locally: bundle exec jekyll serve
5. Create GitHub repo: yourusername.github.io
6. Git push to GitHub
7. Enable GitHub Pages (automatic for .github.io repos)
8. Visit https://yourusername.github.io ✨


💡 Tips
───────

• Employment filtering uses client-side JavaScript
  (No server-side processing needed)

• Profile sidebar is sticky on desktop, responsive on mobile
  (Good UX on all devices)

• Collections make it easy to add employment/projects
  (Just create new .md files in folders)

• All filters run instantly
  (No page reloads, smooth user experience)

• Sample data helps you understand the format
  (Replace gradually as you add your content)
